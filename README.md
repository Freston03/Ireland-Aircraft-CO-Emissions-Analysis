# Ireland Aircraft CO₂ Emissions Analysis

Analysis of aircraft CO₂ emissions over Irish airspace using OpenSky Network data (June 2022).

## Scripts
- `Project_airplane_R.R` – Main analysis and visualisations
- `Project_airplane_R_updated.R` – Updated version
- `Updated_maps.R` – Map visualisations

## Requirements
Uses `renv` for package management. Run `renv::restore()` to install dependencies.
